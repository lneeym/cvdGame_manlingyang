## ğŸŒˆ Why This Game Exists

Real talk: About 300 million people worldwide have color vision deficiency (CVD). Most games either ignore them or treat colorblindness as a difficulty setting. We wanted to make something differentâ€”a game where being colorblind makes you special, not disadvantaged.

This started as a graduate school project exploring empathy through game design. It evolved into something we hope makes both colorblind and non-colorblind players think differently about perception and ability.
